# EUDAT B2FIND (b2find-eudat-eu)

EUDAT B2FIND is a organization open-data portal for European Union running **CKAN**, exposing a standardized catalog API over ~1,466,694 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/b2find-eudat-eu/refs/heads/main/apis.yml)
